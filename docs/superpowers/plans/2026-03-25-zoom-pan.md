# Phase 1.5 — Zoom & Pan Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add scroll-wheel zoom and pan to the canvas, enabling full-resolution inspection of frozen frames.

**Architecture:** A new `viewport.js` module owns zoom/pan state and coordinate transforms. `canvasPoint()` maps screen→image coordinates. Rendering wraps everything in a viewport `ctx.scale/translate`. Each task is designed to maintain a working app — at zoom=1 (the default), behavior is unchanged.

**Tech Stack:** Vanilla JS, Canvas 2D API.

**Spec:** `docs/superpowers/specs/2026-03-25-zoom-pan-design.md`

### Task 1: Create viewport.js Module

**Files:**
- Create: `frontend/viewport.js`

This creates the viewport module with identity-transform defaults. At zoom=1 and pan=0,0,
the transform functions are identity — no visible change to the app.

- [ ] **Step 1: Create the module**

Create `frontend/viewport.js`:

```js
// ── Viewport state for zoom & pan ─────────────────────────────────────────
// All annotation coordinates are in image-space. The viewport transforms
// between image-space and screen-space (canvas pixels on screen).

export const viewport = {
  zoom: 1.0,    // scale factor (1.0 = fit-to-window)
  panX: 0,      // image-space X of viewport's top-left corner
  panY: 0,      // image-space Y of viewport's top-left corner
};

export let imageWidth = 0;
export let imageHeight = 0;

export function setImageSize(w, h) {
  imageWidth = w;
  imageHeight = h;
}

/** Image-space → screen-space (for rendering outside viewport transform) */
export function imageToScreen(x, y) {
  return {
    x: (x - viewport.panX) * viewport.zoom,
    y: (y - viewport.panY) * viewport.zoom,
  };
}

/** Screen-space → image-space (for mouse events) */
export function screenToImage(x, y) {
  return {
    x: x / viewport.zoom + viewport.panX,
    y: y / viewport.zoom + viewport.panY,
  };
}

/** Reset viewport to fit the full image in the canvas */
export function fitToWindow(canvasWidth, canvasHeight) {
  if (imageWidth === 0 || imageHeight === 0) return;
  viewport.zoom = Math.min(canvasWidth / imageWidth, canvasHeight / imageHeight);
  viewport.panX = 0;
  viewport.panY = 0;
}

/** Set zoom to show 1:1 pixels, centered */
export function zoomOneToOne(canvasWidth, canvasHeight) {
  if (imageWidth === 0) return;
  viewport.zoom = 1.0;
  // Center the view
  const visibleW = canvasWidth / viewport.zoom;
  const visibleH = canvasHeight / viewport.zoom;
  viewport.panX = (imageWidth - visibleW) / 2;
  viewport.panY = (imageHeight - visibleH) / 2;
}

/** Clamp pan so the image doesn't scroll completely off-screen */
export function clampPan(canvasWidth, canvasHeight) {
  const margin = 0.1; // 10% overscroll margin
  const visibleW = canvasWidth / viewport.zoom;
  const visibleH = canvasHeight / viewport.zoom;
  const maxPanX = imageWidth - visibleW * (1 - margin);
  const maxPanY = imageHeight - visibleH * (1 - margin);
  viewport.panX = Math.max(-visibleW * margin, Math.min(maxPanX, viewport.panX));
  viewport.panY = Math.max(-visibleH * margin, Math.min(maxPanY, viewport.panY));
}
```

export function resizeCanvas() {
  const r = img.getBoundingClientRect();
  const vr = img.parentElement.getBoundingClientRect();
  canvas.style.left   = (r.left - vr.left) + "px";
  canvas.style.top    = (r.top  - vr.top)  + "px";

### Task 7: Scroll-Wheel Zoom

**Files:**
- Modify: `frontend/main.js`
- Modify: `frontend/render.js` (call resizeCanvas on zoom)

This is where zoom actually starts working!

- [ ] **Step 1: Add wheel event handler**

In `frontend/main.js`, import viewport:
```js
import { viewport, screenToImage, clampPan, fitToWindow, zoomOneToOne, imageWidth, imageHeight } from './viewport.js';
```

Add the wheel handler:

```js
canvas.addEventListener("wheel", e => {
  e.preventDefault();

  // Zoom disabled in live mode
  if (!state.frozen) {
    showStatus("Freeze frame to enable zoom");
    return;
  }

  const rect = canvas.getBoundingClientRect();
  const screenX = (e.clientX - rect.left) * (canvas.width / rect.width);
  const screenY = (e.clientY - rect.top) * (canvas.height / rect.height);

  // Image point under cursor (before zoom)
  const imgPt = screenToImage(screenX, screenY);

  // Zoom in/out
  const factor = e.deltaY < 0 ? 1.15 : 1 / 1.15;
  const newZoom = Math.max(0.5, Math.min(10, viewport.zoom * factor));
  viewport.zoom = newZoom;

  // Adjust pan so the image point stays under the cursor
  viewport.panX = imgPt.x - screenX / viewport.zoom;
  viewport.panY = imgPt.y - screenY / viewport.zoom;

  clampPan(canvas.width, canvas.height);
  resizeCanvas();  // updates canvas resolution for new zoom level
}, { passive: false });
```
